## 🎯 About the Platform

**Code_Tester** is a developer-centric interview environment designed to move technical hiring away from rigid whiteboard memorization and into realistic workflows. 

It pairs a **low-latency collaborative editor** with **integrated video chat** and a **sandboxed multi-language execution engine**, giving candidates a friction-free space to showcase real engineering skills.

---

## 💎 Core Capabilities

| Feature | Description | Built With |
| :--- | :--- | :--- |
| **Real-Time Editor** | Low-latency collaborative canvas with syntax highlighting | React + Monaco Editor |
| **WebRTC Video** | Integrated video and audio channels in the workspace | Stream SDK |
| **Secure Sandbox** | Multi-language remote code execution engine | Isolated Containers |
| **AI Evaluation** | Automagic complexity analysis and hint engine | Inngest + LLM APIS |

## 🛠️ The Stack

<details>
  <summary><b>Frontend Technologies</b></summary>
  <br/>
  
  - **Framework:** React 18 (Vite boilerplate)
  - **Router:** React Router DOM
  - **Styling:** TailwindCSS (v4)
  - **Clients:** Firebase client SDK, Stream Chat & Video client SDK
</details>

<details>
  <summary><b>Backend Technologies</b></summary>
  <br/>

  - **Runtime:** Node.js (ES Modules)
  - **Framework:** Express
  - **Database:** MongoDB & Mongoose ORM
  - **Queues/Workers:** Inngest engine
  - **Auth:** Firebase Admin SDK
</details>
